# Support

## Getting help

For questions and usage help:

- Open a GitHub issue with context, logs, and reproduction steps.
- Include your host OS, VS Code version, and Dev Containers extension version.
- Include relevant `.devcontainer/devcontainer.json` snippets when applicable.

## What this repository supports

This repository focuses on the base image and its included tooling. Downstream project-specific configuration issues may need to be addressed in the consuming repository.
